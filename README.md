# FutShirt Store

**FutShirt Store** é uma aplicação web fictícia desenvolvida para fins acadêmicos, com o objetivo de simular um ambiente de e-commerce voltado para a exibição e organização de camisas de futebol. O projeto não realiza vendas ou entregas reais.

## 🎯 Objetivo

Desenvolver uma aplicação web responsiva e funcional que permita aos usuários visualizar camisas de futebol, acessar detalhes dos produtos, realizar cadastros e simular interações típicas de um sistema comercial, com foco em boas práticas de desenvolvimento front-end.

## 🚀 Funcionalidades

- Listagem de produtos com filtros por time, país, tamanho e faixa de preço  
- Página de detalhes do produto com imagem, descrição e botão de compra fictício  
- Formulários de cadastro de usuário e login  
- Cadastro de produtos (acesso restrito ao administrador)  
- Proteção de rotas com controle de acesso por tipo de usuário 
- Dashboard com gráficos de visualização e vendas simuladas  
- Otimização de desempenho com lazy loading  
- Consumo de Web APIs (JSON-Server, Correios para frete, Power BI para relatórios)

## 🛠️ Tecnologias Utilizadas

- Angular  
- TypeScript  
- JSON-Server  
- Chart.js / Power BI  
- Git & GitHub  
- GitFlow  
- Commits Semânticos
